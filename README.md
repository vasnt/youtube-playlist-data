# YouTube Playlist Data

This repository contains data about my YouTube subscriptions and their top-performing videos.

## Data Format

The `youtube_subscriptions.csv` file includes:

- **Channel Name**: YouTube channel name
- **Video Title**: Full title of the video
- **Video URL**: Direct link to the video
- **View Count**: Total views
- **Like Count**: Total likes  
- **Comment Count**: Total comments
- **Video Age**: How long ago the video was uploaded
- **Published Date**: Exact upload timestamp
- **Category**: AI-assigned content category
